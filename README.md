### Python Built-In Methods

### upper() and lower() methods

The upper() and lower() methods must be called on a string value. They return a new string either in all uppercase or all lowercase.

```
x = 'Abc'
x = x.upper()
>>>'ABC'

x = x.lower()
>>>'abc'
```
