### Python built-in Methods

### isupper() and islower() methods

The isupper() and islower() methods must be called on a string value. They return a new string either in all uppercase or all lowercase.

```
x = 'Abc'
x = x.isupper()
>>>'ABC'

x = x.islower()
>>>'abc'
```
